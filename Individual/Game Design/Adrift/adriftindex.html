Game Design

Adrift

Specifications


  • Genre: Narrative point-and-click adventure game
  • Platform(s): PC / Mobile
  • Engine: Game Concept / Game Design Document
  • Team size: 1
    • Omar Ruiz (Game Design/Game Concept)
  • Production time: 1 week
  • Type: Game concept submission (Project to apply to Cologne Game Lab)

Notable technologies/tools used


  • Prezi To develop the game design document presentation
  • MindMup for storyboarding
  • Aseprite for pixel art

Context of the project


The project was developed under the guidelines to outline a game concept with the (very) abstract topic of 06F. No further details were given. Only those three characters.

Description


Adrift is a short, simple and minimalistic classic point-and-click narrative experience that trades puzzles and action for an immersive, suspensful, atmospheric and exploratory experience.

In an era where most of the games make great efforts in order to transmit a message as clearest as possible or represent reality in the closest possible way, via hyper-realistic graphics, outstanding plausible sound effects, professional voice acting, etc; few are the games that try to prod and stimulate the playerÂ´s interpretation, imaginative and analytic capabilities.

With no dialogues or text at all, and with not so defined pixelated, old-school visual style, Adrift's story is entirely conveyed to the player, only with what they can perceive from the intrinsic game elements alone. This minimalistic approach really helps the purpose of creating a game with few as possible "distractions" or possible pre-conceived author's ideas of what the player is experiencing, in order to demand them a more "maieutic" approach to the game's story.
